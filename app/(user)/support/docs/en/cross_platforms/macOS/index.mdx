# macOS 지원

WatchCon의 macOS 플랫폼 지원에 대해 설명합니다.

## 시스템 요구사항

- macOS 10.15 이상
- Intel 또는 Apple Silicon
- 4GB RAM 이상
- 500MB 저장공간

## 설치 방법

1. 설치 파일 다운로드
2. 패키지 설치
3. 보안 설정
4. 초기 설정

## 주요 기능

- 원격 제어
- 파일 전송
- 화면 공유
- 멀티 모니터 지원
