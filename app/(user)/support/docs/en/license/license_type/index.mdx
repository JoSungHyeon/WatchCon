# 라이선스 종류

WatchCon에서 제공하는 다양한 라이선스 종류와 각각의 특징을 설명합니다.

## 라이선스 유형

1. 개인용 라이선스
   - 단일 사용자용
   - 기본 기능 제공
   - 개인 사용자에게 적합

2. 기업용 라이선스
   - 다중 사용자 지원
   - 고급 관리 기능
   - 기업 환경에 최적화

3. 엔터프라이즈 라이선스
   - 무제한 사용자
   - 커스텀 기능 지원
   - 전문 기술 지원
