# 사용자 인터페이스

WatchCon의 사용자 인터페이스 구성과 사용 방법을 설명합니다.

## 메인 화면

1. 상단 메뉴바
   - 파일 메뉴
   - 편집 메뉴
   - 보기 메뉴
   - 도구 메뉴

2. 사이드바
   - 연결 목록
   - 주소록
   - 채팅
   - 설정

3. 메인 영역
   - 원격 화면
   - 도구 패널
   - 상태 표시줄

## 기능 버튼

- 연결 버튼
- 파일 전송
- 화면 캡처
- 채팅 시작
- 설정 열기

## 단축키

- 연결: Ctrl + N
- 종료: Ctrl + W
- 전체화면: F11
- 설정: Ctrl + ,
