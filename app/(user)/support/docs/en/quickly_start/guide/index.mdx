# 빠른시작가이드

WatchCon을 처음 사용하시는 분들을 위한 빠른 시작 가이드입니다.

## 시작하기

1. 설치하기
   - 프로그램 다운로드
   - 설치 마법사 실행
   - 초기 설정

2. 계정 만들기
   - 회원가입
   - 이메일 인증
   - 프로필 설정

3. 첫 연결
   - WatchCon-ID 확인
   - 상대방 연결
   - 승인 및 시작

## 주요 기능

- 원격 제어
- 파일 전송
- 채팅
- 화면 공유

## 팁과 요령

- 단축키 활용
- 보안 설정
- 성능 최적화
- 문제 해결
