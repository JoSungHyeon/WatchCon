# 일반사항 FAQ

WatchCon 사용과 관련된 일반적인 질문과 답변입니다.

## 자주 묻는 질문

1. WatchCon은 어떤 서비스인가요?
   - WatchCon은 전문적인 원격 제어 및 지원 솔루션입니다.
   - 다양한 플랫폼을 지원하며 안전한 원격 접속을 제공합니다.

2. 어떤 기능들을 제공하나요?
   - 원격 제어
   - 파일 전송
   - 화면 공유
   - 채팅 기능
   - 세션 녹화

3. 무료로 사용할 수 있나요?
   - 기본 기능은 무료로 제공됩니다.
   - 고급 기능은 유료 라이선스가 필요합니다.
