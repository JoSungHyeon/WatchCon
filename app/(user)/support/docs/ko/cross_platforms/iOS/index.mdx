# iOS/iPadOS 지원

WatchCon의 iOS/iPadOS 플랫폼 지원에 대해 설명합니다.

## 시스템 요구사항

- iOS 14.0 이상
- iPadOS 14.0 이상
- 2GB RAM 이상
- 100MB 저장공간

## 설치 방법

1. App Store 다운로드
2. 권한 설정
3. 초기 설정
4. iCloud 연동

## 주요 기능

- 원격 접속
- 파일 공유
- 화면 미러링
- Face ID 인증
